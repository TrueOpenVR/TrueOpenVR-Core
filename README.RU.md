# True Open VR
True Open VR - True Open VR это очень простой API, который позволяет получить доступ ко всем VR устройствам, любых производителей, не требуя, чтобы приложения обладали конкретными знаниями устройства, на которое они нацелены. Этот репозиторий является исходным кодом основной библиотеки, содержащей API.

- Для разработки приложений нужно импортировать функции напрямую из главной библиотеки, вы можете посмотреть [демонстрационные примеры](https://github.com/TrueOpenVR/TrueOpenVR-Samples) или прочитать [документацию](https://github.com/TrueOpenVR/TrueOpenVR-Core/tree/master/Library).

- Для разработки драйверов для VR устройств вы можете взглянуть на [образцы драйверов](https://github.com/TrueOpenVR/TrueOpenVR-Drivers).

- Отправляйте ваши исправления, демонстрационные примеры и так далее.

- Пишите ваши предложение или запрос новой функциональности через "Issue" или почту.
